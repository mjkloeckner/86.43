## Quick Start

Install node modules

```console
$ npm i
```

Create local web server on current directory with `vite`

```console
$ vite
```

Then go to the URL provided by `vite`

> If you don't have `vite` installed you can install it system-wide with the
> following command
> 
> ```console
> # npm -g i vite
> ```
